## Brief
"preliminary-evaluator" is a utility evaluation reference provided by ICCAD'24 Contest Problem B. This tool evaluates the testcase input and the output from contestants' submissions, and provides a preliminary score.
Currently, "preliminary-evaluator" should be executed on Linux machine.
## Usage 
$ ./preliminary-evaluator <testcase> <contestant's_submissions>
## Example
$ ./preliminary-evaluator sanityTestcaseFull.txt sanity_correct.txt
